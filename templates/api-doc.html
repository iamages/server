GET /iamages/api

Renders this API help page.

GET /iamages/api/latest

Returns the latest 10 files.

Response (JSON)

          
            {
              "FileIDs": [1, 2, 3, 4, ...]
            }

Glossary (Response)

  • FileIDs (list): An array containing the FileID of 10 latest files.

Response codes

  • 200: Request completed successfully.

GET /iamages/api/random

Redirects to a random working /iamages/api/info/(FileID: int).

Response codes

  • 307: Redirect to a random file at /iamages/api/info/(FileID: int).
  • 503: No compatible files could be found.

PUT /iamages/api/upload

Uploads a file to Iamages.

Request (JSON)

          
            {
              "UserName": "",
              "UserPassword": "",
              "FileDescription": "",
              "FileNSFW": false,
              "FilePrivate": false,
              "FileExcludeSearch": false,
              "FileData": "",
              
            }

Glossary (Request)

  • UserName (str): The file owner's username (optional).
  • UserPassword (str): The file owner's password (optional).
  • FileDescription (str): The file's description.
  • FileNSFW (bool): The file's NSFW flag.
  • FilePrivate (bool): The file's Private flag (will only be applied with UserName and UserPassword)
  • FileExcludeSearch (bool): The file's exclude from search flag.
  • FileData (str): The file's data, encoded in a base64 string with the file type header stripped.

Response (JSON)

          
            {
              "FileID": ""
            }

Glossary (Response)

  • FileID (int): The file's unique ID.

Response codes

  • 200: Request completed successfully.
  • 400: Missing/malformed request data.
  • 401: User authentication failed.
  • 413: The file's size exceeds the server's configured maximum file size value.
  • 415: The file's MIME type is not supported.

PATCH /iamages/api/modify

Modify file metadata or delete file.

Request (JSON)

          
            {
              "UserName": "",
              "UserPassword": "",
              "FileID": 0,
              "Modifications": {
                "FileDescription": "",
                "FileNSFW": false,
                "FilePrivate": false,
                "FileExcludeSearch": false,
                "DeleteFile": any
              }
            }

Glossary (Request)

  • UserName (str): The file owner's username.
  • UserPassword (str): The file owner's password.
  • FileID (int): The file's unique ID.
  • Modifications (dict): Modifications to apply to the file.
    • FileDescription (str): The file's description.
    • FileNSFW (bool): The file's NSFW flag.
    • FileExcludeSearch (bool): The file's exclude from search flag.
    • FilePrivate (bool): The file's Private flag.
    • DeleteFile (any): Set to anything to delete file.

Response (JSON)

          
            {
              "FileID": 0,
              "Modifications": []
            }

Glossary (Response)

  • FileID (int): The file's unique ID.
  • Modifications (list): Modifications applied to the file.

Response codes

  • 200: Request completed successfully.
  • 400: Missing/malformed request data.
  • 401: User authentication failed.
  • 404: The given file could not be found.

GET /iamages/api/info/(FileID: int)

Returns all publicly available metadata about the given file.
If you have the user's UserName and UserPassword, add the Request (header) in.

Request (URL argument)

          /iamages/api/info/(FileID)

Request (header)

          Authorization: Basic b64encode(UserName:UserPassword)

Glossary (Request)

  • FileID (int): The file's unique ID.
  • b64encode(UserName:UserPassword) (str): The file owner's username and password in UserName:Password format, encoded with base64 using utf-8 encoding.

Response (JSON)

          
            {
              "FileID": 0,
              "FileDescription": "",
              "FileNSFW": false,
              "FilePrivate": false,
              "FileMime": "",
              "FileWidth": 0,
              "FileHeight": 0,
              "FileCreatedDate": "",
              "FileExcludeSearch: false
            }

Glossary (Response)

  • FileDescription (str): The file's description.
  • FileNSFW (bool): The file's NSFW flag.
  • FilePrivate (bool): The file's Private flag.
  • FileMime (str): The file's MIME data.
  • FileWidth (int): The file's width.
  • FileHeight (int): The file's height.
  • FileCreatedDate (str): The file's date of creation in ISO8601 format (UTC time).
  • FileExcludeSearch (bool): The file's exclude from search flag.

Response codes

  • 200: Request completed successfully.
  • 400: Missing/malformed request data.
  • 401: User authentication failed.
  • 404: The given file could not be found.

POST /iamages/api/search

Searches for similar matches to the given search string.

Request (JSON)

          
            {
              "UserName": "",
              "UserPassword": "",
              "FileDescription": ""
            }

Glossary (Request)

  • UserName (str): The files owner's username (optional).
  • UserPassword (str): The files owner's password (optional).
  • FileDescription (str): The search string to be used.

Response (JSON)

          
            {
              "FileDescription": "",
              "FileIDs": []
            }

Glossary (Response)

  • FileDescription (str): The search string used.
  • FileIDs (list): A list of FileIDs that may match the search string.

Response codes

  • 400: Missing/malformed request data.
