GET /iamages/api

Renders this API help page.

GET /iamages/api/latest

Returns the latest 10 files.

Response body type: JSON 

            {
              "FileIDs": [1, 2, 3, 4, ...]
            }

POST /iamages/api/upload

Uploads a file to Iamages.

Request body type: JSON 

            {
              "FileName": "This is a file Name",
              "FileDescription": "This is a file description",
              "FileData": "base64-encoded string with header stripped",
              "UserName": "optional",
              "UserPassword": "optional"
            }


Response body type: JSON 
            {
              "FileIDs": [1, 2, 3, 4, ...]
            }
